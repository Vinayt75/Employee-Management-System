#  Employee Management System (Python & MySQL)

A professional, feature-rich desktop application designed to streamline employee record management. Built with **Python (Tkinter)** for a modern GUI and **MySQL** for robust data persistence.

---

##  Project Description
The **Employee Management System** is a secure, database-driven application that allows organizations to manage their workforce data efficiently. From a secure login gateway to a comprehensive dashboard, this tool automates the process of storing, searching, and updating employee details, replacing traditional manual record-keeping.

##  Key Features
* **Secure Login System:** Protects sensitive data with a dedicated authentication page.
* **Comprehensive Data Fields:** Manage complete profiles including:
    * **Employee ID** (Unique Identifier)
    * **Name** (Full Name)
    * **Phone** (Contact Details)
    * **Role** (Job Designations like Web Developer, Data Scientist, etc.)
    * **Gender** (Male/Female/Other)
    * **Salary** (Payroll Information)
* **Advanced CRUD Operations:**
    * **Add Employee:** Register new staff members instantly.
    * **Update Employee:** Modify existing records with one click.
    * **Delete Employee:** Remove specific records as needed.
    * **Delete All:** A powerful feature to clear the entire database at once (Bulk Management).
* **Smart Search & Filters:** Quickly find any employee using the "Search By" functionality (ID or Name).
* **Real-time Treeview:** A structured table view that updates immediately when data is modified.

##  Tech Stack
* **Frontend:** Python Tkinter (GUI)
* **Backend:** Python
* **Database:** MySQL (Relational Database Management)
* **Libraries:** `mysql-connector-python`, `Pillow` (for Image handling)

##  Project Structure
* `login.py` - The entry point for user authentication.
* `main.py` - The main dashboard containing all management logic.
* `database_setup.sql` - SQL script to initialize the database and tables.
